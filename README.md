in the num.js file, pass the arguments in call of the 'parse()' function in form of the array of arrays

Example:

```
parse([
  [" _  ",
   "|_| ",
   "|_| "]
]);
```

Pass as many arrays as you want.

Run the file with:

### `node num.js`
